void(* d_elu) (const aitensor_t *x, const void *alpha, aitensor_t *result)

Required math function: Derivative of ELU.

Requires a math function that calculates the element wise ELU derivative of a tensor:

\[ result_{i} = \begin{cases} \alpha \cdot e^{x_i} & \text{if } x_i < 0\\ 1 & \text{if } x_i \geq 0 \end{cases} \]

◆ elu

void(* elu) (const aitensor_t *x, const void *alpha, aitensor_t *result)

Required math function: ELU.

Requires a math function that calculates the element wise ELU of a tensor:

\[ result_{i} = \begin{cases} \alpha \cdot (e^{x_i} - 1) & \text{if } x_i < 0 \\ x_i & \text{if } x_i \geq 0 \end{cases} \]
