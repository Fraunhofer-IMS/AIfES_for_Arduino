Detailed Description

Base layer implementation of the ELU activation layer.

This is an "abstract" data-type independent implementation. To use the layer use one of the provided implementations for a specific hardware and data-type (for example from ailayer_elu_default.h) or set the required math functions on your own.

The ELU layer is used as an activation function layer right after a dense layer. It calculates

\[ y = \begin{cases} \alpha (e^x - 1) & \text{if } x \leq 0\\ x & \text{if } x > 0 \end{cases} \]

for every element of the input tensor.

The results of the forward pass of this layer are written to the result tensor of the base ailayer_t struct.

Initialize and connect the given ELU layer.

This function represents the "constructor" of the abstract ELU layer. It initializes the layer structure and connects it to the previous layer.
This function is not intended to call it directly. Instead use one of the data type specific implementations (like for example ailayer_elu_f32_default()).

Parameters
*layerThe layer to initialize.
*input_layerThe previous layer that provides the inputs to the layer.
Returns
Pointer to the (successfully) initialized general layer structure (ailayer_elu.base)

void ailayer_elu_backward ( ailayer_t *  self)

Calculate the backward pass for the given ELU layer.

Implementation of ailayer.backward.

It uses the deltas tensor of the next layer as input and writes the result of the backward pass to the deltas tensor (ailayer.deltas) of the given layer.

Calculation of the errors for the previous layer:

\[ \delta_{in} \leftarrow \delta_{out} \circ ELU'(x_{in}) \]

\( x_{in} \): Result of the forward pass of the previous layer
\( \delta_{in} \): Result of the backward pass of this layer
\( \delta_{out} \): Result of the backward pass of the next layer

Used math functions:

  • ailayer_elu.elu
  • ailayer_elu.d_elu
  • ailayer_elu.multiply
Parameters
*selfLayer to calculate the backward path for.

void ailayer_elu_forward ( ailayer_t *  self)

Calculate the forward pass for given ELU layer.

Implementation of ailayer.forward.

It uses the result tensor of the previous layer as input and writes the result of the forward pass to the result tensor (ailayer.result) of the given layer.

Calculation of the forward pass result:

\[ x_{out} \leftarrow ELU(x_{in}) \]

\( x_{in} \): Result of the forward pass of the previous layer
\( x_{out} \): Result of the forward pass of this layer

Used math functions:

  • ailayer_elu.elu
Parameters
*selfLayer to calculate the forward path for.
